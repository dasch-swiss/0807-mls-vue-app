# mls-vue-app

## Install Yarn if not present

```bash
$ brew install yarn
```

## Project setup

```bash
$ yarn install
```

## Usage

### Compiles and hot-reloads for development

```bash
$ yarn serve
```

### Compiles and minifies for production

```bash
$ yarn build
```

### Run your tests

```bash
$ yarn test
```

### Lints and fixes files

```bash
$ yarn lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
